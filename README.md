# DJUANDA Random User
Random Name library Javascript

## Installation



```bash
//download file and use the file
<script src="Djuanda.js"></script>
```
<br/>
Or
<br/>
```bash
<script src="http://portofolio.hajidalakhtar.site/Djuanda.js"></script>
```


## Usage

```
<script>
    Djuanda.randomNameId('class_Name')
</script>
```
